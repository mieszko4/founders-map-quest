## Set up

### Pre-requirments

* node (0.12.7)
* npm (2.11.3)
* compass
* phantomjs

After cloning, run `npm install` and `bower install`.

## Build & development

Run `grunt` for building and `grunt serve` for preview.

## Testing

Running `grunt test` will run the unit tests with karma.
